### exponentialDecayCalculator.py
* 用来方便计算机器学习里的exponential decay。
* exponential decay公式：**finalValue = initialValue（decayRate）^（globalStep/decayStep）**
* 选择目标，输入其他4个值。
* 获得结果。
* 选择时可用q或Q退出。
